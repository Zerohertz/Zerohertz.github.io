기본 Plot

  • legend('name_1','name_2')로 범례 삽입
  • axis([X_min X_max Y_min Y_max])로 스케일 조정 가능
  • xlabel('name'), ylabel('name')로 축의 이름 지정
  • title('name')로 plot의 이름 지정
  • set(gca,'fontsize',20)으로 폰트의 크기 지정
  • plot.Color(4)로 투명도 지정
